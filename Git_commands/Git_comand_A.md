# คำสั่ง git ที่ขึ้นต้นด้วยอักษร A

1. git add: ใช้ในการเพิ่มการเปลี่ยนแปลงในไฟล์หรือโฟลเดอร์ลงในพื้นที่สเตจ (staging area) เพื่อเตรียมสำหรับการ commit

<img width="496" alt="Screenshot 2023-10-25 112907" src="https://github.com/anndyyzzz/Git_A-Z_Mission_65030276/assets/144866059/962ceeb8-d93f-4a5a-92f4-3e3d82386035">

2. git am: ใช้ในการรับแพทช์ (patch) จากอีเมลหรือไฟล์แพทช์แล้วแปะ (apply) แพทช์นั้นลงใน repository ของคุณ

<img width="508" alt="Screenshot 2023-10-25 112956" src="https://github.com/anndyyzzz/Git_A-Z_Mission_65030276/assets/144866059/46fa8cec-235b-477a-a6e8-0cf91c0428ac">

3. git archive: ใช้ในการสร้างไฟล์เอกสารสำหรับโครงการ Git จากสรา้งงานหรือสาขาที่กำหนด

<img width="504" alt="Screenshot 2023-10-25 113028" src="https://github.com/anndyyzzz/Git_A-Z_Mission_65030276/assets/144866059/2b73e12e-90c7-485d-bee1-d3c936bd1434">

<img width="502" alt="Screenshot 2023-10-25 113035" src="https://github.com/anndyyzzz/Git_A-Z_Mission_65030276/assets/144866059/eac93713-c74b-4516-a589-4168155051d2">

4. git apply: ใช้ในการรับแพทช์ (patch) และแปะ (apply) แพทช์นั้นลงใน repository ของคุณ โดยคล้ายกับ "git am" แต่ใช้แพทช์ที่มีรูปแบบแบบย่อ

<img width="495" alt="Screenshot 2023-10-25 113755" src="https://github.com/anndyyzzz/Git_A-Z_Mission_65030276/assets/144866059/df7a1c6c-1085-42f9-8943-03ee56ee5603">

5. git annotate (หรือ git blame): ใช้ในการแสดงข้อมูลเกี่ยวกับบรรทัดแต่ละบรรทัดในไฟล์ เช่นผู้รับผิดชอบและการอัปเดตล่าสุด

<img width="504" alt="Screenshot 2023-10-25 113909" src="https://github.com/anndyyzzz/Git_A-Z_Mission_65030276/assets/144866059/c05c7a38-b298-44a1-981d-371f8e06414e">

6. git add--interactive: ใช้ในการเพิ่มการเปลี่ยนแปลงในรูปแบบตอน (patch) แบบอินเตอร์แอคทีฟ โดยใช้การให้คำสั่งแบบบรรทัดคำสั่ง
