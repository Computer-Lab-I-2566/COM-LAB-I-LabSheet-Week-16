### ตัวอย่าง

### git add . : เพิ่มทุกไฟล์ใน Directory ปัจจุบัน

### git add -A : เพิ่มทั้งหมดใน Project

### git add *.html : เพิ่มหลายไฟล์โดยระบุนามสกุล

### git archive : สร้างไฟล์เก็บถาวรของ Git

### git am : แปลงข้อความที่แก้ไขแล้วในไฟล์ไปยังการคอมมิต
