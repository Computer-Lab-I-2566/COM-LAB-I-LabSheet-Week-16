# คำสั่ง git ที่ขึ้นต้นด้วยอักษร F

1. git fetch: ใช้เพื่อดึงข้อมูลจากเซิร์ฟเวอร์ Git แต่ไม่รวมรหัสล่าสุดลงในสถานที่ทำงานปัจจุบัน คำสั่งนี้ใช้ในการอัปเดตข้อมูลและข้อมูลรีโมท
