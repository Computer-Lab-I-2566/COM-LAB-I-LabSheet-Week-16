# คำสั่ง git ที่ขึ้นต้นด้วยอักษร H
#### git help คือ คำสั่งที่ใช้สำหรับแสดงความช่วยเหลือเกี่ยวกับคำสั่ง Git ต่างๆ สามารถใช้ได้ทั้งคำสั่งทั่วไปและคำสั่งขั้นสูง

#### รูปแบบการใช้งาน : git help <คำสั่ง>
