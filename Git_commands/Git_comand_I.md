# คำสั่ง git ที่ขึ้นต้นด้วยอักษร I
## git Init
เป็นคำสั่งที่มีไว้ใช้งานที่โฟลเดอร์งานนั้น
## git init --bare
ใช้เพื่อสร้างเรื่องราว Git โดยไม่มีไดเรกทอรีทำงาน, ใช้สำหรับ remote repository
